SEEKPIX is a responsive image search application that allows users to explore high-quality images using the Unsplash API.
The project focuses on delivering a clean and user-friendly interface while efficiently fetching and displaying images based on user search queries.

Features :

1. Search images in real-time
2. High-quality images from Unsplash
3. Clean and minimal UI
4. Fully responsive design
5. Smooth user experience

Tech Stack :

1. React
2. CSS
3. Unsplash API

How It Works :

1. User enters a search query
2. API fetches relevant images from Unsplash
3. Images are displayed dynamically on the screen
